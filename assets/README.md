# Assets

Dossier contenant toutes les ressources statiques de l'application :

- `icons/` → Icônes de l'application (Windows, macOS, Linux)
- `images/` → Images diverses (bannières, illustrations, photos)
- `sounds/` → Sons éventuels (notifications, etc.)

## Règles

- Toujours optimiser les images (TinyPNG, Squoosh, etc.)
- Préférer les formats modernes (WebP + fallback PNG)
- Ne jamais committer de fichiers > 2 Mo sans raison
- Les icônes doivent suivre les spécifications du fichier `icons/README.md`
