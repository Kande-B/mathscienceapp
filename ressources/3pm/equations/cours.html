Chapitre : Équation du premier degré à une inconnue

Le principe de la balance

Une équation est comme une balance à plateaux en équilibre. Tout ce que vous faites d'un côté (ajouter, soustraire, multiplier, diviser), vous devez absolument le faire de l'autre côté pour garder l'équilibre !

I. Qu'est-ce qu'une équation ?

Vocabulaire

  • Une équation est une égalité mathématique comportant au moins une lettre (souvent $x$).
  • Cette lettre s'appelle l'inconnue (c'est le nombre qu'on cherche à trouver).
  • Le signe « = » sépare l'équation en deux parties appelées les membres de l'équation.
  • Résoudre une équation, c'est trouver la valeur de l'inconnue pour laquelle l'égalité est vraie. Cette valeur s'appelle la solution.

Vérifier si un nombre est solution

Pour vérifier si un nombre est la solution, on remplace $x$ par ce nombre et on calcule chaque membre séparément.

Exemple : Le nombre 4 est-il solution de l'équation $3x - 5 = x + 3$ ?

On remplace $x$ par 4 dans le 1er membre : $3 \times 4 - 5 = 12 - 5 = \mathbf{7}$

On remplace $x$ par 4 dans le 2ème membre : $4 + 3 = \mathbf{7}$

Les deux résultats sont égaux (7 = 7), donc 4 est bien la solution de cette équation !

II. Méthodes de résolution

L'objectif final est toujours de se retrouver avec x = un nombre tout seul d'un côté.

Étape 1 : Regrouper les termes (Additions / Soustractions)

On « annule » un terme d'un côté en faisant l'opération inverse des deux côtés. (Astuce : quand un terme change de côté, il change de signe).

Cas $x + a = b$

$x + 7 = 12$

$x + 7 \mathbf{- 7} = 12 \mathbf{- 7}$

$x = 5$

Cas $x - a = b$

$x - 4 = 10$

$x - 4 \mathbf{+ 4} = 10 \mathbf{+ 4}$

$x = 14$

Étape 2 : Isoler x (Multiplications / Divisions)

Si $x$ est multiplié par un nombre, on divise les deux membres par ce nombre. S'il est divisé, on multiplie.

Cas $ax = b$

$3x = 15$

$\frac{3x}{\mathbf{3}} = \frac{15}{\mathbf{3}}$

$x = 5$

Attention aux signes !

$-2x = 8$

$\frac{-2x}{\mathbf{-2}} = \frac{8}{\mathbf{-2}}$

$x = -4$

III. L'équation complète type "ax + b = cx + d"

La méthode en 3 étapes

  1. Regrouper tous les termes en $x$ dans le membre de gauche (ou droite).
  2. Regrouper tous les nombres seuls dans l'autre membre.
  3. Diviser par le nombre qui multiplie $x$.

Exemple complet :     $5x + 3 = 2x - 9$

Étape 1 : on soustrait 2x     $5x - 2x + 3 = -9$

On simplifie     $3x + 3 = -9$

Étape 2 : on soustrait 3     $3x = -9 - 3$

On simplifie     $3x = -12$

Étape 3 : on divise par 3     $x = \frac{-12}{3}$

La solution est : $x = -4$

IV. Mise en équation d'un problème métier

Problème de devis

Un électricien facture un déplacement à 45 € et l'heure de main-d'œuvre à 35 €. Le client paie une facture totale de 150 €. Combien d'heures l'électricien a-t-il travaillé ?

1. Choix de l'inconnue : Soit $x$ le nombre d'heures de travail.

2. Mise en équation : Le prix total est le déplacement + (tarif horaire $\times$ nombre d'heures) :
45 + 35$x$ = 150

3. Résolution :

$35x = 150 - 45$

$35x = 105$

$x = \frac{105}{35}$

$x = 3$

4. Conclusion : L'électricien a travaillé pendant 3 heures.
