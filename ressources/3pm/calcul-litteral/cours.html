Chapitre : Calcul littéral et identités remarquables

Pourquoi apprendre le calcul littéral ?

En atelier ou sur chantier, on utilise des formules avec des lettres (surface, volume, résistance…). Savoir manipuler ces expressions permet de calculer plus vite et d'éviter les erreurs.

I. Le calcul littéral – rappels

Définition

Une expression littérale (ou algébrique) est une expression mathématique qui contient au moins une lettre représentant un nombre (appelée variable).

Exemples : $3x + 2$, $2(a + b)$, $\frac{1}{2} \times b \times h$

Règles de base

  • $a \times b$ s'écrit $ab$ (le signe × est souvent omis).
  • $3 \times x$ s'écrit $3x$ — le chiffre est toujours avant la lettre.
  • $1 \times x$ s'écrit simplement $x$ (le coefficient 1 est implicite).
  • $x \times x$ s'écrit $x^2$ (lire « x au carré »).

Calcul d'une expression — Substitution

Pour calculer la valeur d'une expression, on remplace la lettre par la valeur donnée, puis on effectue les calculs.

Exemple : Calcule $3x - 1$ pour $x = 4$.

$3x - 1 = 3 \times 4 - 1 = 12 - 1 = \mathbf{11}$

Application métier : La formule de la surface d'un rectangle est $S = L \times l$. Calcule $S$ pour $L = 8$ m et $l = 3{,}5$ m.

$S = 8 \times 3{,}5 = \mathbf{28 \text{ m}^2}$

II. Développement et réduction

Définitions

  • Développer : supprimer les parenthèses en utilisant la distributivité.
  • Réduire : regrouper les termes semblables (même lettre, même exposant).

La distributivité simple

C'est la règle de base pour développer :

$k(a + b) = ka + kb$    et    $k(a - b) = ka - kb$

Exemple 1 : $3(x + 5) = 3x + 15$

Exemple 2 : $2(4a - 7) = 8a - 14$

Exemple 3 : $-4(y + 3) = -4y - 12$

Méthode : Développer puis réduire

Développe et réduis : $2(3x + 4) + 5x - 1$

$= 6x + 8 + 5x - 1$   (distributivité)

$= 6x + 5x + 8 - 1$   (on regroupe les termes semblables)

$= 11x + 7$

Attention au signe « – » devant les parenthèses !

$-(3x - 2) = -3x + 2$ (on change tous les signes)

Erreur fréquente : $-(3x - 2) \neq -3x - 2$ ✗

III. Factorisation

Définition

Factoriser une expression, c'est l'opération inverse du développement : on met un facteur commun en évidence pour écrire l'expression sous forme d'un produit.

Méthode : Trouver le facteur commun

  1. Identifier le facteur commun à tous les termes (nombre ou lettre).
  2. Le mettre devant des parenthèses.
  3. Écrire dans les parenthèses ce qui reste après division par ce facteur.
  4. Vérifier en développant le résultat.

Exemple 1 : $6x + 9 = 3(2x + 3)$   (facteur commun : 3)

Exemple 2 : $5a - 15a^2 = 5a(1 - 3a)$   (facteur commun : 5a)

Exemple 3 : $4x + 12 = 4(x + 3)$   (facteur commun : 4)

IV. Les identités remarquables

Définition

Les identités remarquables sont des formules algébriques toujours vraies, qu'il est utile de connaître par cœur. Elles permettent de développer ou de factoriser très rapidement certaines expressions.

Carré d'une somme

$(a + b)^2 = a^2 + 2ab + b^2$

Exemple :

$(x + 3)^2 = x^2 + 2 \times x \times 3 + 3^2 = x^2 + 6x + 9$

$(2a + 5)^2 = (2a)^2 + 2 \times 2a \times 5 + 5^2 = 4a^2 + 20a + 25$

Moyen mnémotechnique : « le carré du premier + 2 × premier × second + le carré du second »

Carré d'une différence

$(a - b)^2 = a^2 - 2ab + b^2$

Exemple :

$(x - 4)^2 = x^2 - 2 \times x \times 4 + 4^2 = x^2 - 8x + 16$

$(3a - 2)^2 = 9a^2 - 12a + 4$

Différence de deux carrés

$(a + b)(a - b) = a^2 - b^2$

Exemple (développement) :

$(x + 5)(x - 5) = x^2 - 25$

Exemple (factorisation) :

$x^2 - 9 = (x + 3)(x - 3)$

$4a^2 - 49 = (2a + 7)(2a - 7)$

Très utile pour factoriser rapidement. Repérez la forme $\square^2 - \square^2$.

V. Application professionnelle

Calcul de surface en carrelage

Un carreleur pose des dalles carrées de côté $(x + 2)$ cm. Il veut connaître la surface d'une dalle en développant l'expression $(x + 2)^2$.

$(x + 2)^2 = x^2 + 2 \times x \times 2 + 2^2 = x^2 + 4x + 4$ cm²

Pour $x = 8$ cm : $S = 8^2 + 4 \times 8 + 4 = 64 + 32 + 4 = \mathbf{100 \text{ cm}^2}$

Vérification : $(8 + 2)^2 = 10^2 = 100$ ✓

Puissance électrique (contexte maintenance)

La puissance dissipée dans une résistance est $P = R \times I^2$. Si la résistance est $R = 50\ \Omega$ et le courant $I = (a + 1)$ A, exprimer $P$ sous forme développée.

$I^2 = (a + 1)^2 = a^2 + 2a + 1$

$P = 50(a^2 + 2a + 1) = 50a^2 + 100a + 50$ W
