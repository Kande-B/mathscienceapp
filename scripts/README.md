# Scripts utilitaires

Ce dossier contient des scripts Node.js pour automatiser des tâches.

## Scripts prévus (à venir)

- `generate-icons.js` — Génère automatiquement les .ico / .icns à partir d'un PNG source
- `validate-formations.js` — Vérifie que le fichier `data/formations.json` est valide
- `backup-data.js` — Sauvegarde automatique des données avant mise à jour

## Utilisation

```bash
node scripts/validate-formations.js
```
